# Tic-Tac-Toe Game (WinForms)

A classic **Tic-Tac-Toe** game built using **C#** and **Windows Forms**. This project was developed as a practical exercise to master UI event handling, game logic implementation, and state management in the .NET Framework.

## 2. Features
* **Interactive UI:** Clean and simple interface using PictureBoxes for the game grid.
* **Two-Player Mode:** Play locally with a friend (Player 1 vs Player 2).
* **Win/Draw Detection:** Automatically detects horizontal, vertical, and diagonal wins, or a draw if the board is full.
* **Game Reset:** Quickly restart the game at any time using the "Restart" button.
* **Visual Feedback:** Clearly displays whose turn it is and highlights the winner.

## 3. Technologies Used
* **Language:** C#
* **Framework:** .NET Framework 4.7.2
* **Platform:** Windows Forms (WinForms)

## 4. How to Run
1.  Ensure you have **Visual Studio** installed.
2.  Clone this repository or download the source code.
3.  Open the `.sln` or `.csproj` file in Visual Studio.
4.  Press `F5` or click **Start** to run the application.

## 5. How to Play
* The game starts with **Player 1**.
* Click on any empty square to place your mark (X or O).
* The game will announce the winner or a draw once the round ends.
* Click **Restart Game** to play again.
